# Topsis_Modeling_Ayush_Kaushik

# 1. Methodology

Text data Collection ----> Finding result using different models ---> Calculation of accuracy and other parameters ----> Generating Topsis Value 

# 2. Description

- Dataset: stanfordnlp/sst2.csv
- Weights: Non-negative (> 0)
- Impacts: Positive/Negative (+/-)

- Applying basic mathematics to fing the topsis score.
